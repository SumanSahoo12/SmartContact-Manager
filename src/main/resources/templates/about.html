About Smart Contact Manager

Smart Contact Manager is a secure, user-friendly web application built using Spring Boot and Thymeleaf. It helps users manage their personal and professional contacts in one place efficiently and securely.

Features

  • ✅ User Registration and Login
  • ✅ Role-based Access Control (Admin/User)
  • ✅ Add, Update, Delete Contacts
  • ✅ Profile Image Upload for Contacts and Users
  • ✅ Search and Filter Contacts
  • ✅ Responsive Bootstrap UI

Technologies Used

  • 💻 Java with Spring Boot (MVC, Security)
  • 🛢️ Hibernate + MySQL
  • 🧩 Thymeleaf Templating Engine
  • 🎨 Bootstrap for Styling
  • 🔁 Git & GitHub for Version Control
